# petarmircheski.com

My personal portfolio website built with Next.js, featuring my blog, photography, and music interests.

## Features

- **Blog** - Personal blog with MDX support for rich content
- **Photos** - Photography gallery with lightbox viewing
- **Music** - Music-related content and interests

## Tech Stack

- [Next.js 15](https://nextjs.org/) - React framework
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [MDX](https://mdxjs.com/) - Markdown with JSX support
- [Content Collections](https://www.content-collections.dev/) - Content management
- [TypeScript](https://www.typescriptlang.org/) - Type safety

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run format` | Format code with Biome |
| `npm run lint` | Lint code with Biome |
| `npm run type-check` | Run TypeScript type checking |
